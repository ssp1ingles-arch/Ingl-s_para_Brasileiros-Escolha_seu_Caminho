# REGRAS — Sistema 09 · Dúvidas Pontuais

> Leia também o `REGRAS_GERAIS.md` na raiz.

## 1. Nome e propósito
**Dúvidas Pontuais.** Gramática sob medida — os pontos específicos que o usuário quis dominar. Cada dúvida vira um card de referência rápida (ex.: `DID`, `WENT`). É o Sistema de respostas cirúrgicas a perguntas concretas.

## 2. O que PERTENCE aqui (exemplos concretos)
- Pontos gramaticais específicos solicitados (`DID` vs `DO`, `WENT`, `used to`, `there is/are`).
- Cada ponto como **card de referência rápida**: regra curta + exemplos + contraste.
- Vocabulário/estruturas garimpados de outros Sistemas quando viram dúvida pontual.

## 3. O que NÃO pertence aqui
- Cursos/progressões completas → **Sistema 01**.
- Material de prática em massa → **Sistema 10**.
- Reduções → **Sistema 02**; verbos por sentido → **Sistema 03**.

## 4. Formato padrão do painel
Painel dark em grade de cards. Cada card = uma dúvida: título + regra objetiva + exemplos (afirmativo/negativo/pergunta quando aplicável). Botão "← Voltar ao início" → `../index.html`.

## 5. Regras específicas
- Cor de acento: laranja `#F97316`.
- Cada card é **autossuficiente e curto** — referência rápida, não aula longa.
- Responde a uma dúvida real; não inventar tópicos que ninguém pediu.
